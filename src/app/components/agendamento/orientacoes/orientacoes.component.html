ORIENTAÇÕES DO GAP-BR event Só é possível agendar para datas dentro da semana atual (Segunda à Sexta). schedule Só é possível agendar a partir de 09:10 da Segunda. repeat Só é possível marcar uma vez a cada 15 dias. access_time A tolerância é de 10 minutos após o horário marcado. assignment O Atendimento será realizado conforme marcação prévia no sistema de agendamento. cancel Só é possível desmarcar com antecedência mínima de 30min.
